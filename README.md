# Система Управления Складом

Простое веб-приложение для учета товаров на складе, разработанное с использованием Flask и SQLite. Позволяет отслеживать приход, расход, текущие остатки и вести базовую аналитику движения товаров.

## Основные возможности

- **Справочники**:
  - **Товары**: Ведение списка товаров с привязкой к категориям и единицам измерения.
  - **Категории**: Группировка товаров по категориям.
  - **Единицы измерения**: Создание и управление единицами измерения (шт, кг, л и т.д.).

- **Складские операции**:
  - **Приход (IN)**: Регистрация поступления товаров на склад с указанием количества и цены закупки.
  - **Расход (OUT)**: Регистрация отгрузки товаров со склада.
  - **Редактирование и удаление операций**: Возможность исправить или отменить ошибочно созданные документы.

- **Учет остатков**:
  - **Текущие остатки**: Просмотр актуального количества каждого товара на складе.
  - **Оценка стоимости**: Автоматический расчет примерной стоимости остатков на основе цен последних поступлений.
  - **Фильтрация и поиск**: Удобный поиск по названию товара, фильтрация по категории и статусу (в наличии / нет в наличии).

- **История и аналитика**:
  - **История операций**: Полный журнал всех приходных и расходных операций с возможностью фильтрации по дате, типу операции и товару.
  - **Статистика**: Просмотр сводной информации по движению товаров за выбранный период.
  - **Детальная аналитика**: Анализ движения и оборота по конкретному товару или целой категории, включая общие объемы поступлений/отгрузок и их стоимость.
  - **FIFO**: При расходе себестоимость списания рассчитывается по методу FIFO (First-In, First-Out), используя партии товаров от самых ранних поступлений.

## Технологии

- **Бэкенд**: Python 3, Flask
- **База данных**: SQLite 3
- **Фронтенд**: HTML, CSS (без использования сложных фреймворков, для простоты)

## Установка и запуск

Для работы приложения требуется установленный Python 3.

1.  **Клонируйте репозиторий**:
    ```bash
    git clone <URL вашего репозитория>
    cd <папка проекта>
    ```

2.  **Установите зависимости**:
    Приложение использует только Flask.
    ```bash
    pip install Flask
    ```

3.  **Запустите приложение**:
    ```bash
    python app.py
    ```
    При первом запуске будет автоматически создан файл базы данных `warehouse.db` с необходимой структурой таблиц.

4.  **Откройте в браузере**:
    Перейдите по адресу [http://127.0.0.1:5000](http://127.0.0.1:5000), чтобы начать работу.

## Структура базы данных

База данных состоит из следующих таблиц:

- `categories`: Справочник категорий.
- `units`: Справочник единиц измерения.
- `products`: Основная таблица товаров.
- `operations`: Заголовочная часть операций (приход/расход).
- `operation_items`: Строки (позиции) в документах операций.
- `lots`: Таблица для учета партий товаров для реализации FIFO. Хранит информацию о цене закупки и остатке в каждой партии.
- `lot_consumptions`: Таблица, связывающая расходные операции с конкретными партиями, из которых был взят товар.
